Small pill for status, type, and category labels — clip types ("Link", "Image"), states ("Pinned", "Local only"), or counts.

```jsx
<Badge tone="secure" dot>Local only</Badge>
<Badge tone="primary">Pinned</Badge>
<Badge tone="neutral" size="sm">200 items</Badge>
```

Tones: `neutral`, `primary`, `secure`, `accent`, `success`, `warning`, `danger`, `solid`. Use `dot` for a status indicator, `icon` for a leading glyph.
