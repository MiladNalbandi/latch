Surface container for grouped content — settings sections, feature tiles, marketing cards.

```jsx
<Card>Default paper card</Card>
<Card variant="raised" interactive onClick={open}>Clickable feature</Card>
<Card variant="ink" padding="lg">Dark panel</Card>
```

Variants: `default`, `raised` (shadow, no border), `flat`, `ink` (dark surface). `padding`: `none`/`sm`/`md`/`lg`. Add `interactive` for hover-lift.
